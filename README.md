# Pyrshell
	Pyrshell is a simple shell, that is being developed for me to get a better handle on rust

## Prerequisites
	Rust-Lang 1.13.0
	Cargo 0.13.0

## Getting Started
	Clone the repo
	cd to repo
	run "Cargo build"
	run ./target/debug/shelly

## Installing
	Clone repo
	cd to repo
	run "Cargo build"
	copy target/debug/pyrshell to /usr/bin
	run "exec pyrshell"
